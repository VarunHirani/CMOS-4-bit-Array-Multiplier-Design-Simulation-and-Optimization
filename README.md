# CMOS 4-bit Array Multiplier: Design, Simulation and Optimization

## 📌 Overview

This project implements a **4-bit array multiplier** using CMOS logic and evaluates its performance through SPICE simulations. The design is built hierarchically starting from basic logic gates up to a complete multiplier, followed by optimization for delay, power, and area.

---

## ⚙️ Project Objectives

* Design CMOS logic gates (AND, OR, XOR)
* Implement a Full Adder using basic gates
* Build a 4-bit ripple carry adder
* Construct a 4-bit array multiplier
* Analyze:

  * Propagation delay
  * Power consumption
  * Transistor area
* Optimize the multiplier design

---

## 🏗️ Design Flow

1. **Basic Gates Design**

   * CMOS implementation of AND, OR, XOR
   * VTC analysis
   * Propagation delay measurement

2. **Full Adder Design**

   * Built using logic gates
   * Delay characterization

3. **4-bit Adder**

   * Ripple carry architecture
   * Critical path delay estimation

4. **Array Multiplier**

   * Based on partial product generation and addition
   * Functional verification via waveforms

5. **Optimization**

   * Improved design for:

     * Reduced delay
     * Lower power consumption
     * Reduced area

---

## 📊 Results

* Functional correctness verified through SPICE waveforms
* Critical path delay calculated for:

  * Base design
  * Optimized design
* Power and area comparison included
